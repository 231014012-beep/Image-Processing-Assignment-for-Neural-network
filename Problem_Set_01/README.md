# Problem Set 01 – CNN Medical Image Classification

## Objective

The goal is to build a CNN model.
The model classifies chest X-ray images.
It predicts whether the image is Pneumonia or Normal.

## Dataset

Total images: 5863
Two classes: Pneumonia and Normal

## Methodology

* Used a CNN model
* Rescaled the images
* Used three convolution layers
* Added dropout to reduce overfitting
* Trained the model for 5 epochs

## Results

accuracy: 0.7580
loss: 2.1574
Test Accuracy: 0.7580128312110901

## Conclusion

The CNN model can classify pneumonia and normal X-ray images successfully.
