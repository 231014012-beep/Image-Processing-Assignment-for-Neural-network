# Problem Set 02 – Bank Term Deposit Prediction

## Objective

Build a Logistic Regression model to predict whether a customer will subscribe to a term deposit.

## Dataset

Bank Marketing dataset containing 17 attributes of customer banking behavior.

## Methodology

1. Loaded dataset
2. Explored data
3. Converted categorical variables to numeric
4. Split dataset into training and testing sets
5. Trained Logistic Regression model
6. Evaluated using accuracy, confusion matrix, and classification report

## Results

* Accuracy ≈ 90%
* Model predicts customer subscription behavior

## Conclusion

Logistic Regression model helps identify customers likely to subscribe to term deposits.
